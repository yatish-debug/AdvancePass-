# AdvancePass v5.0

An enterprise-grade AI-powered Password Security & Audit Platform.

## Features
- Dashboard with real-time analytics
- Password Strength Checker
- Password Generator
- Bulk Audit from CSV
- AI Advisor Placeholder

## Running Locally

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Run the application:
   ```bash
   streamlit run frontend/streamlit_app.py
   ```

## Deployment
Can be deployed using Docker or directly to Streamlit Community Cloud.
